7. Query complete consultation data for email:
```bash
supabase db remote shell -c "
SELECT
  c.consultation_code,
  p.full_name as patient_name,
  p.email as patient_email,
  d.full_name as doctor_name,
  TO_CHAR(c.start_time, 'Month DD, YYYY at HH24:MI') as consultation_date,
  c.diagnosis,
  c.summary,
  pr.doctor_notes,
  pr.prescription_code,
  pr.email_sent_at IS NOT NULL as email_sent
FROM larinova_consultations c
JOIN larinova_patients p ON c.patient_id = p.id
JOIN larinova_doctors d ON c.doctor_id = d.id
LEFT JOIN larinova_prescriptions pr ON pr.consultation_id = c.id
WHERE c.status = 'completed'
ORDER BY c.created_at DESC
LIMIT 3;
"
```

8. Get prescription items for email template:
```bash
supabase db remote shell -c "
SELECT
  p.prescription_code,
  pi.medicine_name,
  pi.dosage,
  pi.frequency,
  pi.duration,
  pi.instructions
FROM larinova_prescriptions p
JOIN larinova_prescription_items pi ON p.id = pi.prescription_id
WHERE p.id = (SELECT id FROM larinova_prescriptions ORDER BY created_at DESC LIMIT 1)
ORDER BY pi.created_at;
"
```

9. Check email sent status:
```bash
supabase db remote shell -c "
SELECT
  prescription_code,
  CASE
    WHEN email_sent_at IS NOT NULL THEN 'Sent'
    ELSE 'Pending'
  END as email_status,
  TO_CHAR(email_sent_at, 'YYYY-MM-DD HH24:MI:SS') as sent_at,
  TO_CHAR(created_at, 'YYYY-MM-DD HH24:MI:SS') as created_at
FROM larinova_prescriptions
ORDER BY created_at DESC

"
```

10. Generate email statistics:
```bash
supabase db remote shell -c "
SELECT
  COUNT(*) as total_prescriptions,
  COUNT(email_sent_at) as emails_sent,
  COUNT(*) - COUNT(email_sent_at) as emails_pending,
  ROUND(COUNT(email_sent_at)::numeric / COUNT(*)::numeric * 100, 2) as success_rate
FROM larinova_prescriptions;
"
```
