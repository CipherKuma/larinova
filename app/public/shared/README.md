# Shared Brand Assets

The canonical Larinova logo sources live in `ops/brand/logos/`.

Use explicit filenames for new code:

- `dark-mode-icon-only.png`
- `dark-mode-icon-text.png`
- `light-mode-icon-only.png`
- `light-mode-icon-text.png`

Legacy aliases such as `larinova-icon.png` and `larinova-logo-dark.png` are kept
only for existing code paths. Their contents must remain synced from the
canonical files above.
